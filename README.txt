This is a restaurant management appliaction, where administrators can register menu items and waiters, waiters can be assigned to tables that they can assign a number of customers and then take orders, and at the end they can calculate the total price. 

If you wish to test this program, I reccomend deleting all the files inside the next folder:

ProyectoEstDDat\\ProyectoEstDDat\\Info\\

This project was developed as a class asignment for the Data Structures class in the "Facultad de Ciencias Fisico Matemáticas" in "Universidad Autónoma de Nuevo León" in the JAN-JUN 2019 semester. 

This project was showcased in the Expo LMAD of June 2019.
